Newton Method(뉴턴법) 개념 설명

생성일: 2026-02-22 | 카테고리: 수학/알고리즘
수치해석 최적화 알고리즘 수학

요약

뉴턴법은 비선형 방정식 f(x)=0의 해를 구하는 강력한 수치해석 알고리즘입니다. "현재 위치에서 접선을 그어 x축과의 교점을 다음 추정값으로 삼는" 단순하지만 강력한 아이디어로, 2차 수렴 속도를 가지며 현대 과학/공학 계산의 핵심 도구로 사용됩니다.

1) 배경과 문제의식

방정식의 해를 구하는 것은 수학과 과학의 가장 근본적인 문제 중 하나입니다. 특히 비선형 방정식 f(x) = 0의 해를 구하는 것은 대부분의 경우 대수적으로 정확한 해를 구할 수 없습니다.

예를 들어:

  • x³ - 2x - 5 = 0
  • cos(x) = x
  • e^x = 3x

이런 방정식들은 √2 = 1.414... 처럼 간단한 공식으로 표현할 수 없습니다. 17세기 아이작 뉴턴은 "근사적으로 해에 점점 가까워지는 방법"을 고안했고, 이것이 현대 수치해석의 핵심 도구가 되었습니다.

핵심 문제의식: "정확한 해를 모르더라도, 반복적으로 개선하여 원하는 정확도까지 빠르게 근사할 수 있을까?"

2) 제1원리(First Principles)

Newton Method는 접선 근사(tangent line approximation)라는 간단한 아이디어에서 출발합니다.

기본 원리

  1. 시작점 선택: 해에 가까울 것 같은 초기 추정값 x₀를 선택
  2. 접선으로 근사: 현재 점 (x_n, f(x_n))에서 함수를 1차 테일러 전개로 선형 근사
  3. 접선과 x축의 교점: 이 접선이 x축과 만나는 점을 다음 근사값 x_(n+1)로 사용
  4. 반복: 수렴할 때까지 2-3 과정 반복

수학적 공식

뉴턴 반복식(Newton's iteration formula):

x_(n+1) = x_n - f(x_n) / f'(x_n)

유도 과정:

  • 점 (x_n, f(x_n))에서의 접선 방정식: y = f(x_n) + f'(x_n)(x - x_n)
  • 이 접선이 x축(y=0)과 만나는 점: 0 = f(x_n) + f'(x_n)(x_(n+1) - x_n)
  • 정리하면: x_(n+1) = x_n - f(x_n)/f'(x_n)

기하학적 해석

그래프에서 보면:

  1. 현재 점에서 함수의 접선을 그림
  2. 그 접선이 x축과 만나는 지점 = 다음 추정값
  3. 곡선이 접선보다 더 빨리 꺾이므로, 보통 실제 해에 더 가까워짐

수렴 조건

  • f'(x) ≠ 0 (미분값이 0이 아님)
  • 초기값이 해에 충분히 가까움
  • 2차 수렴(quadratic convergence): 조건이 좋으면 반복할 때마다 정확한 자릿수가 약 2배씩 증가

vs. 할선법(Secant Method)

  • 할선법: 미분값 대신 두 점을 잇는 할선 사용
  • 공식: x_(n+1) = x_n - f(x_n) · (x_n - x_(n-1)) / (f(x_n) - f(x_(n-1)))
  • 미분을 모를 때 유용하지만, 뉴턴법보다 수렴이 느림 (약 1.618차 수렴)

vs. 고정점 반복법(Fixed-point Iteration)

  • 뉴턴법은 고정점 반복법의 특수한 경우
  • g(x) = x - f(x)/f'(x)로 설정한 x = g(x)의 고정점 반복
  • 뉴턴법은 더 빠른 수렴을 위해 최적화된 형태

4) 맥락적 연결

다차원 확장: Newton-Raphson Method

연립방정식 시스템으로 확장:

x_(n+1) = x_n - J⁻¹(x_n) · F(x_n)

여기서 J는 야코비 행렬(Jacobian matrix)

최적화 문제

  • 최솟값/최댓값 찾기: f'(x) = 0을 풀면 됨
  • f'(x) = 0에 뉴턴법 적용 → x_(n+1) = x_n - f'(x_n)/f''(x_n)
  • 이것이 뉴턴 최적화 알고리즘의 기초

머신러닝 연결

  • 경사하강법(Gradient Descent): 1차 미분만 사용 (느림)
  • 뉴턴법: 2차 미분(Hessian) 사용 → 훨씬 빠르지만 계산 비용 높음
  • 준뉴턴법(Quasi-Newton): BFGS, L-BFGS 등 - Hessian 근사로 타협

실제 응용

  • 공학: 비선형 회로 해석, 구조 해석
  • 컴퓨터 그래픽스: 레이 트레이싱의 교점 계산
  • 금융: 옵션 가격의 내재변동성(implied volatility) 계산
  • 물리 시뮬레이션: 비선형 동역학 방정식
  • 신경망 학습: 2차 최적화 알고리즘

한계와 개선

  1. 발산 문제: 초기값이 나쁘면 발산
    • 해결: Line search, Trust region 방법
  2. 미분 계산: 복잡한 함수는 미분이 어려움
    • 해결: 자동 미분(Automatic Differentiation), 유한차분
  3. 다중근: f'(x) = 0인 근처에서 문제
    • 해결: Modified Newton Method

5) [용어 사전]

비선형 방정식 (Nonlinear Equation)

x의 1차식이 아닌 방정식. x², sin(x), e^x 등이 포함된 방정식. 일반적으로 공식으로 정확한 해를 구할 수 없음.

수치해석 (Numerical Analysis)

컴퓨터를 이용해 수학 문제의 근사 해를 구하는 학문 분야.

근사 (Approximation)

정확한 값 대신 충분히 가까운 값으로 대체하는 것. 예: π ≈ 3.14159

접선 (Tangent Line)

곡선 위의 한 점에서 곡선에 딱 붙어서 지나가는 직선. 그 점 근처에서 곡선을 가장 잘 근사함.

1차 테일러 전개 (First-order Taylor Expansion)

함수를 그 점에서의 값과 기울기만으로 선형 근사하는 것: f(x) ≈ f(a) + f'(a)(x-a)

미분 / 도함수 (Derivative)

f'(x): 함수의 기울기. 뉴턴법에서는 접선의 기울기를 구하는 데 사용.

수렴 (Convergence)

반복할수록 목표값(여기서는 정확한 해)에 점점 가까워지는 것.

발산 (Divergence)

반복했을 때 목표값에서 점점 멀어지거나 무한대로 가는 것.

2차 수렴 (Quadratic Convergence)

오차가 제곱에 비례해서 줄어듬. 반복 1번당 정확한 자릿수가 약 2배로 증가. 매우 빠른 수렴.

이분법 (Bisection Method)

구간을 반으로 나누면서 해를 찾는 방법. 느리지만 안정적.

할선법 (Secant Method)

미분 대신 두 점을 잇는 직선(할선)을 사용하는 방법. 뉴턴법보다 느리지만 미분 불필요.

야코비 행렬 (Jacobian Matrix)

여러 변수 함수의 모든 1차 편미분을 모은 행렬. 다변수 뉴턴법에서 사용.

헤시안 (Hessian)

2차 편미분들을 모은 행렬. 최적화 문제에서 곡률 정보 제공.

경사하강법 (Gradient Descent)

함수의 기울기 반대 방향으로 조금씩 이동하여 최솟값을 찾는 방법. 머신러닝의 기본.

고정점 반복법 (Fixed-point Iteration)

x = g(x) 형태로 바꿔서 x_(n+1) = g(x_n)을 반복하는 방법.

결론

뉴턴법은 "현재 위치에서 접선을 그어 x축과의 교점을 다음 추정값으로 삼는" 단순하지만 강력한 아이디어로, 현대 과학/공학 계산의 핵심 도구입니다. 2차 수렴 속도로 빠르게 정확한 해에 도달할 수 있으며, 최적화, 머신러닝, 공학 등 다양한 분야에서 응용되고 있습니다.
